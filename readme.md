# GigaUno

## Tiiviste

GigaUno on kuin Uno steroideilla: Siinä on unon säännöt, mutta erikoiskortteja ja muita lisäsääntöjä on tullut lisää.
Peli on tehty Javalla ja siihen tulee yksinkertainen käyttöliittymä Swingillä.
